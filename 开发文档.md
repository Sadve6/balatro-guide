## 一、需求来源

用户是 Balatro（单机扑克游戏）的玩家，希望做一个网站，能够：
1. 展示全部 150 张 Joker 小丑牌
2. 包含中文名、卡牌效果、Tier 排名（S/A/B/C/D）
3. 提供搜索功能
4. 提供组合构筑攻略
5. 支持手机端访问

---

## 二、技术方案选型

### 方案确定
- **纯前端单文件**：不依赖后端，直接 GitHub Pages 托管
- **数据源**：从另一个公开 Balatro 网站（xiaowanmichelle-wq/balatro-guide）的 `data.js` 提取 150 张 Joker 完整数据
- **中文汉化**：合并已有中文名、Tier 评分、组合Tips数据
- **卡牌图形**：无真实游戏图片，用 SVG 程序生成游戏风格卡面图

### 文件结构
```
balatro-guide-new/
├── index.html      # 主页面（三标签：卡牌/搜索/攻略）
├── data.js        # 150张Joker数据
├── app.js         # 渲染逻辑+交互
├── styles.css     # 深色主题样式
└── jokers_full.json  # 合并后完整数据
```

### 第二阶段：网站开发（2026-03-27）

## 五、关键代码示例

### SVG 卡牌生成（app.js）

```javascript
function generateSVG(j, size) {
    var r = j.rarity || 'common';
    var c = RARITY_COLORS[r];
    var t = j.tier || 'C';
    var nm = j.name_cn || j.name || '';
    var ltr = nm.charAt(0) || '?';

    return '<svg viewBox="0 0 140 200">' +
        '<rect fill="' + c.bg + '" rx="12" width="140" height="200"/>' +
        '<circle cx="70" cy="52" r="26" fill="' + c.border + '"/>' +
        '<text x="70" y="60" text-anchor="middle" font-size="22" fill="white">' + esc(ltr) + '</text>' +
        '<text x="70" y="88" text-anchor="middle" font-size="9.5" fill="white">' + esc(nm.substring(0,6)) + '</text>' +
        '<text x="70" y="170" text-anchor="middle" fill="white">T' + t + '</text>' +
        '</svg>';
}
```
